This repository provides the official implementation of the paper:

“Query Attention: An Effective Plug-and-Play Module for Enhanced Small Dataset Classification.”
Our QA module consists of two key components—Self-Attention-Driven Channel Attention (SACA) and Channel–Spatial Interaction Attention (CSIA)—and can be seamlessly integrated into both CNN and ViT backbones.


📌 Key Features
	•	Plug-and-play design — independent of backbone architecture.
	•	Works for both CNNs and ViTs — improves small-dataset classification performance.
	•	Two main components:
	•	SACA: explicitly models channel dependencies
	•	CSIA: integrates channel-guided global spatial information
	•	Attention-based implementation
	•	Compatible with various datasets (CIFAR-10/100, Tiny-ImageNet, CINIC-10)
